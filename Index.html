Tutorial de como subir una Web a Github

Paso 1

Lo primero que debemos hacer es crearnos una cuenta en Github, a continuacion lo que debemos realizar es un nuevo repositorio el cual compartiremos con nuestro compañero de trabajo.

Paso 2

El segundo paso es descargarnos Github Desktop donde crearemos nuestro depositorio que esta en la nube ha una carpeta local de nuestro propio ordenador.

Paso 3

Mas tarde creamos el index que seria nuestra pagina web, la cual estamos utilizando para explicar este mismo tutorial, y el css con el cual añadiremos el estilo que queramos a la pagina web.

Paso 4

Despues de tener todo creado, hemos tenido que ir subiendo cada commit que nosotros queramos para que nuestro compañero pueda acceder a la informacion de la pagina. Los commits los adjuntamos en una foto aqui abajo.
